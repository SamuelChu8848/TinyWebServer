# TinyWebServer
项目是基于 Linux 的轻量级多线程 Web 服务器，应用层实现了一个简单的HTTP 服务器，支持用户登录与静态资源访问。

* 利用同步**I/O方式模拟 Proactor 模式**，采用**线程池模式**、**非阻塞 LT**的**Epoll**技术，实现服务器的并发模型；
* 使用**状态机**解析HTTP请求报文，支持解析**GET**请求
* 实现非活跃连接的定时检测，完成服务器压力测试，实现**上万的并发连接**数据交换。

快速运行，编译后运行，并指定端口号。
```
g++ *.cpp -pthread
./a.out 10000
```
